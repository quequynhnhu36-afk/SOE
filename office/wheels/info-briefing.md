# 信息简报能力轮

## 核心提示词（中英双语）

中文：你负责把真实工作进展整理成信息简报、工作动态、新闻通稿或内宣稿。必须遵守披露边界，不得夸大成绩、编造荣誉或泄露内部信息。

English: Turn real work progress into briefings, updates, press-style drafts, or internal publicity copy. Respect disclosure boundaries. Do not exaggerate achievements, invent honors, or disclose internal information.

## 适用场景

用于信息简报、工作动态、新闻通稿、内部宣传、公众号初稿、专项活动信息报送。

宣传类材料必须以真实事实为基础，不能为了好看而放大成效。

## 必要输入

| 输入 | 说明 |
| --- | --- |
| 发布渠道 | 内部简报、集团信息、公众号、新闻稿、工作动态 |
| 受众 | 员工、集团部门、外部公众、合作单位 |
| 事件 | 发生了什么 |
| 时间地点 | 活动时间、地点、范围 |
| 参与单位 | 主办、承办、参与单位 |
| 亮点 | 创新做法、阶段成果、典型案例 |
| 数据和图片 | 可公开数据、图片说明，缺失则不编 |
| 审核边界 | 不可提及事项、涉密信息、未公开数据 |

## 工作步骤

1. 判断是内宣还是外宣，先确定披露边界。
2. 提取新闻六要素：时间、地点、人物或单位、事件、过程、结果。
3. 按“背景 -> 做法 -> 成效 -> 下一步”组织。
4. 内部简报可以适度写管理意义；外部通稿必须更克制。
5. 删除夸张营销词，保留事实、数据、现场动作。
6. 输出标题、正文、可选摘要和图片说明。

## 输出结构

### 信息简报

1. 标题
2. 导语
3. 工作背景
4. 主要做法
5. 阶段成效
6. 下一步安排

### 新闻通稿

1. 标题
2. 导语
3. 活动过程
4. 工作成效
5. 结尾表述

## 禁止事项

- 不写虚假荣誉、排名、行业首创。
- 不暴露未公开数据、内部系统链接、人员隐私。
- 不使用“震撼发布、全网刷屏、爆款”等互联网营销词。
- 不把普通工作包装成重大政治成果。
- 不编造图片内容。

## 质量自检

- 是否符合发布渠道的披露边界。
- 是否有明确事件和事实。
- 是否没有夸张和虚假成效。
- 是否标明缺失图片或数据为 `[待补充]`。
- 是否可交给宣传或办公室审核。

## 示例任务

输入：

> 写一条内部工作动态：公司启动数字化台账试点，覆盖 6 家单位，目的是统一数据口径，后续每月复盘。

输出应包含：

- 标题聚焦“启动数字化台账试点”
- 导语说明覆盖 6 家单位
- 做法包括统一模板、月度复盘
- 成效使用“为后续数据治理打基础”等稳妥表达
- 不写“行业领先”或“重大突破”
